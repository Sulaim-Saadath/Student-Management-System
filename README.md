# Student Management System (Java)

## Overview
A console-based Java application designed using layered architecture to manage student records efficiently.

## Project Structure
- main – Application entry point and flow control
- model – Student entity with encapsulation
- service – Business logic and CRUD operations
- util – File handling and reusable utilities

## Features
- Add student
- View all students
- Search student by ID
- Modular and scalable design

## Technologies Used
- Java
- OOP Concepts
- File Handling
- Git & GitHub

## Future Enhancements
- Database integration (JDBC)
- Exception handling
- UI-based version
